Python for Chatbots

In Python for AI, one use of AI mentioned was for chatbots. Such chatbots are possible using Python.

What is a chatbot?

A chatbot is a virtual assistant that interacts with the user through text. In Python, these chatbots can use basic print() and input() commands, or much more detailed processes. In this page, I will go over the process of making a basic chatbot.

While this is useful in this case, it could be a lot cooler, enter Transformers, a library that makes it easy and efficient to make chatbots.

Transformers utilizes AI as well as Natural Language Processing to make chatbots. In order to use Transformers you must have TensorFlow, or any of the other compatible frameworks, installed, as well as a dataset. Note that if no dataset is provided, then Transformers will automatically download one.
